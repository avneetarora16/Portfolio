Project Overview
This was our individual project for the . We were required to find or create a dataset that is of personal interest to you, that had at least 100 records (objects, rows), each with at least 4 fields (properties, columns). Next we had to Create an interactive tabular representation of that dataset. Also, we had to create an interactive graphical visualization of that dataset. To create the interactive tabular representation and graphical visualization, we could use one or multiple libraries from: Tabulator.js, List.js, d3.js, svg.js, p5.js and Underscore.js. I created my interactive tables using Tabulator.js and graphical visualization using d3.js.
